# 🌸 Flower Shop – React Application

A simple **Flower Shop** project built using **React.js**.  
When the user clicks on a flower button (like Rose or Lily), the corresponding flower image is displayed on the screen.

---

## 🚀 Features

- Button-based flower selection
- Displays images dynamically
- Simple and beginner-friendly UI
- Uses React state and event handling
- Built with React + Vite

## 📸 How the App Works

- Click **Rose** → Rose image appears  
- Click **Lily** → Lily image appears  
- Click another flower → That flower image is shown  
- Only one image is displayed at a time
