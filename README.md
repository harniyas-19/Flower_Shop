# 🌸 Flower Shop – React Application

A simple Flower Shop project built using React.js.

## 📌 Description
This application displays flower images dynamically.
When a user clicks a flower button (Rose, Lily, etc.), the corresponding image is shown.

## 🚀 Features
- Button-based flower selection
- Dynamic image rendering
- Simple and clean UI
- Built with React + Vite

## 🛠️ Technologies Used
- React.js
- JavaScript
- CSS
